# Báo Cáo Document Generator (cong-cu-tao-bcdxcv) - Repository Context

## Tổng Quan Dự Án
Đây là một ứng dụng Web-based B2B (Mô hình SaaS) dùng để tự động thiết kế, map dữ liệu và xuất trực tiếp các báo cáo, tài liệu Word (.docx) chuyên nghiệp (định hướng phục vụ Ban lãnh đạo, báo cáo P/E, cổ phiếu, v.v.).

Hệ thống cho phép nhập/xuất file, import template word, tự động nhận diện Placeholder từ Word, xây dựng một Catalog biến (Field Catalog) trực quan trên web, hỗ trợ gán (map) danh sách dữ liệu linh hoạt bằng cả API nội bộ lẫn công việc Data Entry trực tiếp.

## Tech Stack
- **Framework:** Next.js 16 (App Router) + React 19
- **Ngôn ngữ:** TypeScript, Python (Data Pipeline)
- **CSS / UI:** Tailwind CSS v4, Lucide React (Icons)
- **Cơ sở dữ liệu & Storage:** Hybrid Storage Model (Prisma ORM + Better SQLite3 cho Khách hàng) và Local File-system JSON Storage trong thư mục `report_assets/` (Cho Mapping Engine).
- **Xử lý Word (Docx):** `@eigenpal/docx-js-editor` (trình soạn thảo lõi tích hợp mạnh mẽ trên trình duyệt web), `docxtemplater`, `jszip` (đi kèm script Python xuất file ngầm).
- **Xử lý Excel/CSV:** `xlsx`
- **Xác thực dữ liệu:** `zod`

## Tính năng / Modules chính

### 1. Mapping Configuration (Quản lý Data Mapping) - `/report/mapping`
Trái tim của ứng dụng. Giao diện trực quan cho phép:
- Chọn khách hàng (Customer) để map data.
- Kế thừa các "Field Template" đã lưu, hoặc tạo Template từ đầu (Create from scratch).
- Gom nhóm các trường dữ liệu (Grouping / Subgrouping) thành đường dẫn (ví dụ: `Ban lãnh đạo / Cơ cấu lãnh đạo`).
- Nhập/chỉnh sửa giá trị manual theo nhiểu kiểu dữ liệu: Chữ (`string`), Số (`number`), Phần trăm (`percent`), Ngày tháng (`date`), Bảng tuỳ biến (`table`).
- **🌟 Nhóm Lặp (Repeater / Collection):** Hỗ trợ mô hình 1-N (một Group có nhiều Record, ví dụ: Ban lãnh đạo có nhiều thành viên). Cho phép thêm/xóa thẻ (Card) bản ghi linh hoạt để sau này fill bằng thẻ `{#ban_lanh_dao}...{/ban_lanh_dao}` trong file Word mẫu.
- Nút Import cấu trúc file Excel (`.csv`, `.xlsx`).
- Module Validate cấu trúc và Technical keys tự động chống vỡ layout Word.

### 2. Template Editor (Quản lý Mẫu Báo Cáo Docx) - `/report/template`
- Sử dụng lõi `@eigenpal/docx-js-editor` (đã nâng cấp bản v0.0.12) để xem/chỉnh file Word trực tiếp trên web cực nhanh và độc lập.
- Hỗ trợ người dùng mở Menu bên trái chọn Copy các cú pháp Placeholder `[ten_truong]` được bóc tách từ Field Catalog để rải vào file Word hiện tại, lưu lại trực tiếp mà ko cần tải về.

### 3. Customer Data Management - `/report/customers`
- Thêm, sửa, xóa danh sách công ty/khách hàng.

### 4. Background Data Pipeline & Logs - `/report/runs`
- Hệ thống gọi/spawn các script Python ngầm (`run_pipeline.py`, `export_template_stub.py`) để sinh file báo cáo `.docx` thật, tính toán Validate dữ liệu.
- Quản lý nhật ký (Logs) và kết quả của các lần chạy tạo báo cáo.

## Cấu trúc thư mục (Tóm lược)
- `report_assets/`: (QUAN TRỌNG) Thư mục kho chứa lõi Hybrid JSON File-system storage bao gồm: File Mapping Master, Alias Map, Draft Versions, Metadata và kho file Docx export.
- `*.py` (vd: `export_template_stub.py`, `run_pipeline.py`): Các script chạy Data Pipeline Python phục vụ việc tạo file Word. 
- `src/app/api/`: Các endpoint API RESTful nội bộ xử lý file, JSON storage, và Prisma DB.
- `src/app/report/`: Giao diện chính của Dashboard.
  - `mapping/page.tsx`: Core engine quản trị Field Catalog khổng lồ theo dạng bảng lưới có Tree Group.
  - `mapping/components/`: Chứa các Component tái sử dụng như `MappingSidebar.tsx` và `FieldRow.tsx`.
  - `mapping/helpers.ts`: Các hàm Text Utils, Validate Type, xử lý TV không dấu, Slug.
  - `template/page.tsx`: Tab tải lên và preview danh sách File Word.
- `src/components/docx-template-editor-modal.tsx`: Popup Modal chứa lõi Editor từ thư viện `@eigenpal`.
- `src/lib/report/config-schema.ts`: Core Schema định nghĩa toàn bộ mô hình JSON của Field Catalog cho ứng dụng (tích hợp Zod).
- `prisma/schema.prisma`: Schema database SQL cục bộ.
