# UI/UX Review — BCDXCV Banking Tool

**Date:** 2026-03-24 | **Reviewer:** Claude (code-based audit) | **Stack:** Next.js + Tailwind v4

---

## Overall Assessment: 7/10

Design system tốt (semantic tokens, dark mode, consistent palette). Cần cải thiện accessibility, error handling UX, và một số pattern chưa nhất quán.

## Issues (Cần fix)

### P1 — Accessibility (CRITICAL)

| # | Issue | Location | Recommendation |
|---|-------|----------|----------------|
| 1 | **No `aria-live` on error messages** | `customer-detail-view.tsx:255` — error chỉ có visual (text đỏ), không announce cho screen reader | Thêm `role="alert"` vào error `<p>` |
| 2 | **Metadata generic** | `layout.tsx:22-23` — title "Create Next App", description "Generated by create next app" | Đổi thành tên app thật: "BCDXCV — Công cụ tạo hồ sơ vay" |
| 3 | **No skip link** | `layout.tsx` — không có skip-to-content link cho keyboard users | Thêm skip link ẩn đầu body |
| 4 | **Tab buttons thiếu `aria-selected`** | `customer-detail-view.tsx:267-280` — tab buttons dùng visual-only active state | Thêm `role="tablist"`, `role="tab"`, `aria-selected` |
| 5 | **Back button dùng ký tự ←** | `customer-detail-view.tsx:248` — `← Danh sách KHCN` dùng Unicode arrow | Dùng Lucide `ArrowLeft` icon + `aria-label` |

### P2 — Interaction & Feedback (HIGH)

| # | Issue | Location | Recommendation |
|---|-------|----------|----------------|
| 6 | **Save success flash quá nhanh** | `customer-detail-view.tsx:224` — `setTimeout(() => setSaved(false), 2000)` — 2s có thể miss | Tăng lên 3s hoặc dùng toast notification |
| 7 | **Error messages không có recovery** | `customer-detail-view.tsx:255` — chỉ show text error, không có "Thử lại" | Thêm retry button hoặc link hướng dẫn |
| 8 | **DocxPreviewModal thiếu loading indicator** | `docx-preview-modal.tsx` — `next/dynamic` load async nhưng không có loading fallback | Thêm `loading: () => <Spinner />` vào `dynamic()` |
| 9 | **Customer list search không debounce** | `customer-list-view.tsx:36` — filter chạy mỗi keystroke | Thêm debounce 300ms cho search input |

### P3 — Consistency & Polish (MEDIUM)

| # | Issue | Location | Recommendation |
|---|-------|----------|----------------|
| 10 | **Mixed button styles** | Across components — có chỗ dùng `bg-gradient-to-r from-violet-600 to-fuchsia-600`, có chỗ dùng `bg-violet-600` plain | Tạo shared button variants (primary gradient, secondary outline) |
| 11 | **Tab style không nhất quán** | `customer-detail-view.tsx` — main tabs dùng underline style, sub-tabs dùng pill/chip style | OK nếu intentional (hierarchy), nhưng nên document |
| 12 | **Landing CTA link chỉ đến KHDN** | `page.tsx:59` — `href="/report/khdn"` — user KHCN không có entry point | Thêm 2 CTA hoặc route đến dashboard chung |
| 13 | **Empty `<></>` wrapper** | `docx-template-editor-modal.tsx:371` — `<><EigenpalDocxEditor/></>` thừa | Xóa Fragment wrapper |

### P4 — Performance (LOW)

| # | Issue | Location | Recommendation |
|---|-------|----------|----------------|
| 14 | **Landing SVG animations luôn chạy** | `page.tsx:16-35` — 12+ animated elements, không check `prefers-reduced-motion` | Thêm `@media (prefers-reduced-motion: reduce)` |
| 15 | **No `Suspense` boundaries** | Pages load toàn bộ trước khi render | Thêm Suspense cho heavy sections (financial analysis, doc checklist) |

---

## Quick Wins (Dễ fix, impact cao)

1. **Fix metadata** → 1 phút, professional ngay
2. **Thêm `role="alert"`** cho error messages → 2 phút, accessibility compliance
3. **Thêm `loading` cho dynamic imports** → 2 phút, better perceived performance
4. **Fix back button arrow** → 1 phút, consistency với Lucide icons
